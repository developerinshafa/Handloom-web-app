# Handloom E-commerce Platform
A simple and user-friendly Handloom E-commerce Website where users can browse, explore, and purchase traditional handloom products online.
 
## The system supports main users.
 1. Guest User
 2. Registered/Login User
 3. Admin User
 
## Guest User
 A guest user can access the website without logging in.

1. View homepage:
 Guest arrives, browses handloom products (sarees, fabrics, home decor)
2. Browse handloom products(Sarees, Top, Material,etc..)
3. Search and filter items
4. View product details
5. Add items to cart:
 Add items to a temporary session-based cart.
6. Purchases.

## Registered/Login User
 A registered user who logs into the system.
 
1. All guest user features/Authentication: 
 Login vianame, email and phone number.
2. Save address:
 Multiple shipping addresses for fast checkout.
3. Checkout and place orders:
 Automatic pre-filling of shipping and billing information.
4. View order history
 1. Order confirmation
 2. Order history
 3. Order tracking (Processing, Shipped, Delivered)
5. Track orders:
  Order tracking, easy return requests, and loyalty rewards. 

  ## Admin User
  This user type focuses on managing inventory, authenticity, and fulfilling orders.
  
  1. Admin Dashboard:
     Login to a specialized dashboard.
  2. Product Management:
     1. List new handloom items, upload photos, and set prices.
     2. Add detailed descriptions (weaving techniques, time taken).
     3.Update inventory levels to ensure accuracy, as handloom items are often unique or low-stock.
  3.  Order Fulfillment:
     1. View pending orders and customer details.
     2. Update order status (e.g., Packing, Shipped, Delivered).
  4. Communication:
     Receive alerts for new orders and interact with the platform admin for support    
  
## Comparison of Users Table
This section compares the workflows of different user types in the Handloom E-commerce platform.

| Feature              | Guest User | Login User     | Admin User   |
| :-------             | :----------| :------------- | :------------|
| View Products        |  Yes       |   Yes          |  Yes         |
| Add / Edit Products  |  No        |   No           |  Yes         |
| Search / Filter      |  Yes       |   Yes          |  No          |
| Add to Cart          |  Yes       |   Yes          |  No          |
| Register / Login     |  Yes       | Already Logged |  No          |
| Track Orders         |  No        |   Yes          | Manage Orders|
| Dashboard Access     |  No        |   No           | Full Access  |
